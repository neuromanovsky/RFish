# Concept 03 Notes

## Working title

One Perfect Screen

## Цель

Не решать все приложение сразу, а найти один экран, который может стать
камертоном для всей системы.

## Принципы

- только главный экран
- один центральный объект
- минимум вторичного UI
- тихая, дорогая материальность
- настоящие обложки и настоящие треки

## Намерение

Если это направление попадет по тону, из него уже можно будет выращивать
playlist, catch и news.
