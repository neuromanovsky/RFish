# Rare Fish Prototype

Тут подготовлен легкий браузерный каркас под дизайн мобильного радио-приложения.

## Что уже есть

- единый mobile frame для iPhone и Android
- экранная навигация внутри браузерного прототипа
- базовые карточки, плеерный блок, список эфиров и коллекция
- структура, которую можно быстро менять без сборки и зависимостей

## Структура

- `index.html` — точка входа
- `prototype/styles.css` — весь визуальный слой
- `prototype/app.js` — экраны, мок-данные и интерактивность
- `design-direction.md` — зафиксированные принципы визуального направления
- `product-scope.md` — список экранов и функциональный объем прототипа
- `reference-dashboard.md` — рабочий дашборд продуктовых и предметных референсов
- `archive/concepts/` — архив неудачных или отложенных дизайн-концептов
- `archive/directions/` — архив отложенных направлений экранов
- `assets/reference-tracks/` — реальные треки и локально сохраненные обложки
- `Inbox/` — сюда удобно складывать скриншоты и референсы от клиента

## Как открыть

Самый простой вариант:

1. Открыть `index.html` в браузере

Если нужен локальный сервер:

1. Перейти в `/Users/romanovsky/RFish`
2. Запустить `python3 -m http.server 4173`
3. Открыть `http://127.0.0.1:4173`

## Что менять, когда придут материалы

- в `prototype/app.js` заменить названия экранов, тексты и логику переходов
- в `prototype/styles.css` поменять визуальный язык, цвета, типографику и компоненты
- в `Inbox/` держать входящие скриншоты и контент для сверки
